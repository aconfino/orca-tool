You will need to download the sass gem to be able to 
convert sass (Syntatically Awesome Style Sheet) into standard css

http://sass-lang.com/tutorial.html - sass quick start
http://rubyinstaller.org/download.html - ruby installer for windows

For windows users, use the Ruby style backslashes in your command arguments
sass --watch C:/css/sass/style.scss:C:/css/style.css
